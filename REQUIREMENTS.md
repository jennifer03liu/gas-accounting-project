# GAS HTML 郵件設定頁面修改需求

## 主要修改項目：

### 1. 移除特定的項目符號
- 只移除「管理中心小幫手」這一行前面的項目符號
- 保持其他項目符號不變（使用 "•"）
- 管理中心小幫手連結保持在原位置
### 2. 特殊格式標記系統
在 textarea 中使用特殊標記，寄信時自動轉換：

格式標記對照表：
- **重要**文字內容**重要** → 黃底紅字效果
- **粗體**文字內容**粗體** → 粗體效果  
- [顯示文字](網址) → 超連結
- 換行符號 → HTML 換行

### 3. 新增詳細格式說明區塊
在每個內文 textarea 下方加入格式說明，包含：
- 特殊格式標記的使用方法和效果預覽
- 自動替換變數的說明
- 視覺化的格式對照表
- 使用提示和注意事項

### 5. 後端處理函數需求
需要在 GAS Code.gs 中添加文字轉 HTML 的處理函數，用於：
- 轉換重要標記為黃底紅字
- 轉換粗體標記  
- 轉換超連結
- 轉換換行符號

### 6. CSS 樣式優化
- 增加 textarea 的高度到 300px
- 添加格式說明區塊的樣式
- 改善整體視覺效果
- 增加格式說明區塊的藍色左邊框和背景色

### 7. 錯誤處理增強
- 為所有 Google Apps Script 調用添加錯誤處理
- 添加載入超時保護機制  
- 改善錯誤訊息顯示

### 8. 格式說明區塊的具體內容
包含以下元素：
- 📝 標題圖示
- 特殊格式標記的實際效果展示
- 自動替換變數說明
- 💡 使用提示
- 背景色為淺藍色 (#f0f8ff)
- 左邊框為藍色 (#4285F4)

## 預期效果：
用戶在 textarea 中看到純文字格式便於編輯，但實際寄出的郵件會自動轉換成具有黃底紅字、粗體、超連結等格式的美觀 HTML 郵件。
